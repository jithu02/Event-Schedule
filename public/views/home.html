Calendar App

A simple app developed using the JavaScript with MongoDB, Express, AngularJS, and Node.js.

What this app does:
  • Allows you to add new events as well as edit them.
  • Allows you to search for events by either event's title or date range.
  • Validates all events on the UI before they're persisted. The validation process will make sure that the required fields are filled in with valid data.
  • Allows you to remove events.
